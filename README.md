# Lorcana Collection Local

Lorcana Collection Local is a standalone collection management application designed for Disney Lorcana collectors.

The application allows users to organize, track, and manage their collections entirely on their own device without requiring an online account.

Features include:

* Collection tracking
* Set browsing
* Card quantity management
* Wishlist management
* Deck building
* Collection statistics
* Card search and filtering
* Duplicate tracking
* Trade list management
* Local data storage
* Collection value tracking

Lorcana Collection Local is designed for collectors who want full control of their collection data while maintaining a fast and easy-to-use interface.
